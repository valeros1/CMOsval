## Реализованные этапы

### Этап 1: Минимальный прототип с конфигурацией
- Реализовано CLI-приложение с настройкой через аргументы командной строки
- Поддержка всех требуемых параметров:
  - Имя анализируемого пакета
  - URL репозитория или путь к тестовому файлу
  - Режим работы с тестовым репозиторием
  - Версия пакета
  - Имя выходного файла с изображением
  - Режим вывода в формате ASCII-дерева
  - Максимальная глубина анализа
  - Подстрока для фильтрации пакетов
- Валидация параметров и обработка ошибок

### Этап 2: Сбор данных
- Реализована логика получения данных о зависимостях для Maven пакетов
- Поддержка формата пакетов Java (Maven)
- Получение информации о прямых зависимостях из репозитория Maven Central
- Парсинг POM-файлов для извлечения зависимостей

### Этап 3: Основные операции
- Реализовано построение полного графа зависимостей с помощью **BFS с рекурсией**
- Обработка циклических зависимостей
- Поддержка режима тестирования с файловым репозиторием
- Фильтрация пакетов по подстроке
- Ограничение глубины анализа
